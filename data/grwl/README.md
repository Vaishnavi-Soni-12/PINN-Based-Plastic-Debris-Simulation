# data/grwl

## Purpose

Stores local GRWL shapefile components used to derive river width statistics.

## Typical Files

- `GRWL_summaryStats.shp`
- `GRWL_summaryStats.dbf`
- `GRWL_summaryStats.shx`
- `GRWL_summaryStats.prj`
- auxiliary shapefile index files (`.sbn`, `.sbx`, `.cpg`, `.xml`)

## Consumed By

- `data_validation/analysis/width_statistics.py`

## Version Control

Raw shapefile datasets should not be committed because of size and distribution constraints.
