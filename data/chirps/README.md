# data/chirps

## Purpose

Stores local CHIRPS precipitation NetCDF datasets used by offline climate scaling validation scripts.

## Expected File

- `chirps-v2.0.2022.monthly.nc` (or equivalent CHIRPS monthly product)

## Consumed By

- `data_validation/analysis/chirps_validation.py`

## Version Control

Raw NetCDF files are large and should not be committed.
