# data/hydrosheds

## Purpose

Stores local HydroSHEDS global raster layers used for hydrological context and offline validation.

## Subfolders

- `flow_direction_15s/`
  - flow direction raster and HydroSHEDS technical documentation.
- `flow_accumulation_15s/`
  - flow accumulation raster and HydroSHEDS technical documentation.

## Version Control

HydroSHEDS rasters are very large and should remain local-only, not committed to git.
