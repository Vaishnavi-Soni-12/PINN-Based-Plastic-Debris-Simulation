# data_validation/outputs

## Purpose

Stores generated JSON outputs produced by offline validation scripts.

## Typical Files

- `width_ranges.json`
- `climate_scaling.json`
- `slope_ranges.json` (if generated)
- `discharge_ranges.json` (if generated)

## Lifecycle

1. Generated by scripts in `data_validation/analysis/`.
2. Reviewed manually.
3. Promoted selectively into `data/static/`.

## Version Control

Treat this folder as generated output and exclude it from regular commits.
